# 1-网页基础信息
1、网页基础信息
--------

### 什么是HTML

HTML是一种超文本标记语言(Hyper Text Markup Language)

### web标准

web标准是有w3c组织和其他标准组织制定的合计，w3c也叫（万维网联盟）

#### web标准的构成

1.  结构（Struectrue）-- 网页元素
2.  表现（Presentstion）-- 样式
3.  行为（Behavior）-- 交互 js

### html框架

#### 基本框架

```html
<html>
    <head>html头部</head>
    <body>html主体</body>
</html>
```

#### 进阶框架

```html
<!DOCTYPE html> <!-- 加上了这行代码，浏览器就会按照 W3C 的标准规范（标准模式）来渲染页面-->

<html lang="zh-CN"> <!-- 这行代码是用于设置页面的语言，zh-CN 表示中文 en表示英文-->
<head>
    <meta charset="UTF-8"> <!-- 这行代码是用于设置页面的字符编码的，UTF-8 是一种通用的字符编码，可以支持世界上大多数的文字 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 这行代码是用于响应式布局的，可以让页面在不同设备上都能正常显示 -->
    <title>Document</title> <!-- 这行代码是用于设置页面的标题的，会显示在浏览器的标签页上 -->
</head>
<body>
    主要内容
</body>
</html>
```
