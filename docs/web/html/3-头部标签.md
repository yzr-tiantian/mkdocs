# 3-头部标签
### 文档类型声明标签

```html
<!DOCTYPE>
```

*   文档类型声明，作用就是告诉浏览器使用哪种HTML版本来显示网页。
*   这句代码的意思是：当前页面采取的是HTML5版本来显示网页

#### 注意：

1.  `<!DOCTYPE>` 声明位于文档中的最前面的位置,处于标签之前。
2.  `<!DOCTYPE>` 不是一个HTML 标签，它是文档类型声明标签。

* * *

### lang 语言

```html
<html lang="en">
```

用来定义当前文档显示的语言。

1.  `en` 定义语言为英语
2.  `zh-CN` 定义语言为中文
3.  `zh-cmn-hans` 简体中文,更加严谨

简单来说,定义为en就是英文网页,定义为zh-CN就是中文网页

* * *

### charset字符集

```html
<meta charset="UTF-8">
```

作用是告知浏览器当前网页的字符集使用 UTF-8 编码

*   `charset` 是 character set 的缩写,字符集
*   常见的字符集有 GBK 和 UTF-8
*   GBK 是中国的中文字符集
*   UTF-8 是国际通用的字符集

* * *

### 移动端适配

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

作用:设置视口,方便移动端网页的显示

*   `width=device-width` 设置视口宽度为设备宽度
*   `initial-scale=1.0` 设置初始缩放比例为1
*   `user-scalable=no` 禁止用户缩放
*   `maximum-scale=1.0` 设置最大缩放比例为1
*   `minimum-scale=1.0` 设置最小缩放比例为1

* * *

### title 标题

```html
<title>网页标题</title>
```

作用:定义网页的标题,在浏览器标题栏显示

*   `<title>` 标签只能定义一个,在多个 `<title>` 中,浏览器只会显示最后一个
