# 1-css的3种位置
### css基础知识

CSS（Cascading Style Sheets，层叠样式表）。

是用来控制网页在浏览器中的显示外观的语言。CSS3现在已被大部分现代浏览器支持，而下一版的CSS4仍在开发中。

### css 的三种写法

1.  内联样式，写在html标签内部

```html
<p style="color: red;">This is a paragraph.</p>
```

1.  内部样式表，写在html标签内部

```html
<head>
<style>
p {
  color: red;
}
</style>
</head>
```

1.  外部样式表，写在单独的css文件中，然后在html文件中引入

```html
<head>
<link rel="stylesheet" type="text/css" href="#root/uhI7ZB2bsRaq">
</head>
```
