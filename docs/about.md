# 关于

## 项目信息

这是一个使用 MkDocs 和 Material 主题构建的文档网站示例。

## 技术栈

- **MkDocs**: 快速、简单的静态网站生成器
- **Material Design**: Google 的设计语言
- **Python**: 项目基于 Python 构建
